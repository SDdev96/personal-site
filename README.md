- `frontend/` - Applicazione React con TypeScript e Vite

## Come Avviare il Frontend

1. Assicurati di avere [Node.js](https://nodejs.org/) (versione 18 o superiore) installato sul tuo computer.

2. Accedi alla cartella del frontend:
   ```bash
   cd frontend
   ```

3. Installa le dipendenze:
   ```bash
   npm install
   ```

4. Avvia il server di sviluppo:
   ```bash
   npm run dev
   ```

5. Apri il tuo browser e vai a [http://localhost:5173](http://localhost:5173)

## Script Disponibili

- `npm run dev` - Avvia il server di sviluppo
- `npm run build` - Crea una build di produzione
- `npm run lint` - Esegue il linting del codice
- `npm run preview` - Avvia un server di anteprima della build di produzione

## Requisiti di Sistema
